개발자 포털
검증 포털
API
Step 1 · GitHub 활동 분석
오픈소스 기여 역량 분석
GitHub 사용자명을 입력하면 커밋, PR, 코드 리뷰 등 공개 활동을 분석해 역량 점수를 산출합니다.
Step 2 · 검증 가능한 자격증명 발급
Verifiable Credential 발급
분석 결과를 Ed25519 서명으로 봉인한 W3C VC를 발급합니다. 발급된 VC는 서버에 저장되며 누구든 서명을 검증할 수 있습니다.
Step 3 · 블록체인 앵커링 (선택)
Polygon Amoy Testnet에 해시 기록
VC의 SHA-256 해시를 Polygon Amoy 스마트 컨트랙트에 기록합니다. 이후 서버가 없어도 온체인에서 발급 사실을 독립적으로 증명할 수 있습니다.
Step 4 · VC 파일 저장
검증서 다운로드
발급된 VC 문서를 JSON 파일로 저장합니다. 채용담당자나 제3자에게 전달하면 검증 포털에서 즉시 확인할 수 있습니다.
오픈소스 기여 검증서 확인
개발자로부터 전달받은 VC 파일을 업로드하거나 credential ID를 입력하면 서명·무결성·온체인 기록을 즉시 검증합니다.
📄
VC 파일을 여기에 드래그하거나 클릭해서 선택
.json 파일
또는 Credential ID로 조회 (같은 서버)
