검증 포털
API
01 · 분석
오픈소스 기여 역량 분석
GitHub 공개 활동을 AI로 분석하여 도메인 적합도, 기여자 점수, Skill Evidence, 성장 추이를 산출합니다.
02 · 발급
Verifiable Credential 발급
분석 결과를 Ed25519로 서명한 W3C VC를 발급합니다.
03 · 앵커링 (선택)
Polygon Amoy 온체인 기록
VC 해시를 스마트 컨트랙트에 기록합니다. 이후 서버 없이도 독립 검증이 가능합니다.
04 · 저장
검증서 다운로드
VC 문서를 JSON으로 저장합니다. 검증 포털에 업로드하면 즉시 확인 가능합니다.
오픈소스 기여 검증서 확인
개발자로부터 전달받은 VC 파일을 업로드하거나 Credential ID를 입력하면 Ed25519 서명, 무결성, Polygon 온체인 기록을 즉시 검증합니다.
